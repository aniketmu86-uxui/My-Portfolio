AVTOOLS: Integrated macOS Post-Production Suite

The Problem

Fragmented media workflows required editors to switch between multiple heavy applications for basic compression, noise reduction, and project setup, causing severe operational lag.

The Execution

Engineered a standalone macOS application using Python, PyTorch, and DeepFilterNet3. Features an automated Project Launcher that enforces standardized folder structures and metadata tracking across the entire team.

Insight: Centralized tooling enforces standard operating procedures at the software level.
  • DeepFilter AI Audio Cleanup
  • Automated Folder Architecture
  • Standalone macOS Deployment
sys.id_02

Brand-Automated Subtitle Engine

The Problem

Manual subtitling was incredibly slow and highly prone to human error. Revisions frequently drifted from strict corporate brand guidelines regarding font sizing, safe zones, and background opacity.

The Execution

Built a Python/FFmpeg-based pipeline that hard-burns .srt/.vtt files with locked corporate brand constants (Arial, 80% opacity bars). Features automatic scaling from 4K to 720p without manual adjustment.

Insight: Hard-coding brand constants mitigates compliance risks at the render stage.
  • 100% Brand Compliance
  • 4K → HD Auto-Scaling
  • Zero Manual Typesetting
sys.id_03

Intelligent Video Revision Tracker

The Problem

Stakeholder feedback arrived as chaotic, unstructured prose across multiple email threads. This "Feedback Hell" created severe bottlenecks, un-tracked statuses, and increased the risk of missed client notes.

The Execution

Developed a custom JavaScript parser that utilizes RegEx to instantly convert unstructured feedback into a categorized, sortable dashboard. Features priority tagging, "Blocked" status tracking, and 1-click Excel export.

Insight: Transforming unstructured prose into structured data accelerates project velocity.
  • Real-Time Executive Dashboard
  • RegEx Pattern Matching
  • 1-Click Excel Export
